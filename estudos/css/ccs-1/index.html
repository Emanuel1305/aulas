Programação WEB

Programação web é a arte e a ciência de criar sites e aplicativos para a internet. Ela desempenha um papel fundamental na nossa experiência online, permitindo que interajamos com páginas da web, realizemos compras, acessemos informações e muito mais.

HTML

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar e exibir conteúdo na web. Ela é uma das tecnologias básicas para a construção de páginas web e está presente na maioria dos sites

CSS

O CSS foi desenvolvido pelo W3C (World Wide Web Consortium) em 1996 para resolver um problema: o HTML não foi projetado para ter tags que ajudariam a formatar a página. Tags como < font> foram introduzidas na versão 3.2 do HTML e causaram muitos problemas para os desenvolvedores. O CSS veio para resolver essa questão. CSS (Cascading Style Sheets) é um padrão que define como os dados são apresentados no navegador. Enquanto o HTML fornece informações sobre a estrutura de um documento, o CSS indica o aspecto que ele deve ter: a fonte, o fundo, o texto, as cores dos links, as margens e a disposição dos objetos na página

JAVA SCRIPT

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web. Sempre que uma página da web vai além de simplesmente mostrar informação estática e inclui conteúdo que se atualiza em intervalos de tempo, mapas interativos ou gráficos 2D/3D animados, é provável que o JavaScript esteja envolvido
